PickBazar - Grocery Delivery Website
Overview
A modern, responsive grocery delivery landing page with a clean design and user-friendly interface. The page features a navigation bar with brand logo, grocery category selection, and a search functionality for finding grocery items.

Brand Logo: "PickBazar" with highlighted "Pick" text

Grocery Category: Quick access to grocery section

Offer Section: Special deals and promotions

Help Support: Customer assistance with icon

Language Selection: English with dropdown indicator

Join Button: Redirects to Google (external link)

🔍 Main Section
Headline: "Groceries Delivered in 90 Minute"

Subtitle: "Get your healthy foods & snacks delivered at your doorsteps all day everyday"

Search Bar:

Category filter (Grocery)

Search input field

Search button with Amazon integration
